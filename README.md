# React Three Fiber + TypeScript + Vite Starter Template

A very basic react-three-fiber starter template with TypeScript and Vite.

## Installation

```bash
npm ci
```

## Usage

```bash
npm run dev
```

Open [localhost:5173](http://localhost:5173) in your browser
and start editing the App.tsx file in the src directory.
